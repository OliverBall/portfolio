Your primary task as a web developer will be to create reliable and high-performing applications and services, which can be accessed over the internet. Focusing solely on the underlying software and databases (known as the 'back end') is most common, however some web developers work on the interface and visual design (the 'front end'), while others combine both ('full-stack development'). Salary Salaries for junior or entry-level web developers can range from £19,000 to £25,000. Mid-level and senior web developers usually earn between £25,000 and £35,000. Lead developers typically earn between £35,000 and £60,000, potentially rising to £75,000 in London. We prefer our applicants to have some sort of degree or quilifications in either computer science, software engineering or web design and development, without the correct qualifications there are some courses you can take to apply for this role.
